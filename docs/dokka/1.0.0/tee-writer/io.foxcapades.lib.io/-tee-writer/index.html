class TeeWriter(writers: Writer) : Writer

Tee Writer

All writes and actions performed on this TeeWriter are performed on all of the configured writers.

If an exception is thrown by one or more of the underlying writers when performing an action, the action will still be attempted on all other underlying writers before the first exception is rethrown by this writer.

Example

var writer = TeeWriter(stdout, fileWriter1, fileWriter2, logWriter)

reader.copyTo(writer)
